cartesianLine

PURPOSE

CARTESIANLINE Create a straight line from cartesian equation coefficients

SYNOPSIS

function line = cartesianLine(varargin)

DESCRIPTION 

CARTESIANLINE Create a straight line from cartesian equation coefficients

   L = cartesianLine(A, B, C);
   Create a line verifying the Cartesian equation:
   A*x + B*x + C = 0;

   See also:
   lines2d, createLine
