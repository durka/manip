inCircle

PURPOSE

INCIRCLE test if a point is located inside a given circle.

SYNOPSIS

function b = inCircle(point, circle)

DESCRIPTION 

INCIRCLE test if a point is located inside a given circle.

   B = inCircle(POINT, CIRCLE) 
   return true if point is located inside the circle

   Example:
   inCircle([1 0], [0 0 1])
   inCircle([0 0], [0 0 1])
   returns true, whereas
   inCircle([1 1], [0 0 1])
   return false

   See also:
   circles2d, onCircle
