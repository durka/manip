createHomothecy

PURPOSE

CREATEHOMOTHECY Create the the 3x3 matrix of an homothetic transform

SYNOPSIS

function trans = createHomothecy(point, ratio)

DESCRIPTION 

CREATEHOMOTHECY Create the the 3x3 matrix of an homothetic transform

   TRANS = createHomothecy(POINT, K);
   POINT is the center of the homothecy, K is its factor.
