drawVector3d

PURPOSE

DRAWVECTOR3D Draw vector at a given position

SYNOPSIS

function varargout = drawVector3d(pos, vect, varargin)

DESCRIPTION 

DRAWVECTOR3D Draw vector at a given position

   drawVector3d(POS, VECT)

   Example
     figure; hold on;
     drawVector3d([2 3 4], [1 0 0]);
     drawVector3d([2 3 4], [0 1 0]);
     drawVector3d([2 3 4], [0 0 1]);
